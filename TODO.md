# term-diff 実装タスク

## ステータス凡例
- [ ] 未着手
- [x] 完了

## タスク一覧

- [x] 1. ブロック行揃え（alignment）
      削除/追加ブロックが複数行にわたる場合、左右で同じ高さになるよう空白行を挿入して揃える

- [x] 2. 中央ガター（変更マーカー）
      左右パネルの間に `<<` / `>>` の矢印を表示し、変更ブロックの対応関係を示す

- [x] 3. インライン差分ハイライト
      変更行の中でさらに単語/文字レベルで何が変わったかをハイライト

- [x] 4. 変更件数カウント表示
      ヘッダーに「N differences」のような差分ブロック数を表示

- [x] 5. 横スクロール
      長い行が画面幅を超えた場合に横スクロール対応（現状は `…` で切り捨て）
